# ncmdump
Convert .ncm files to .mp3 .flac files.  
Optimized interface fork
## Dependencies
pycryptodome
## Usage
- Install dependency `pip install pycryptodome`
- Create a directory 'in' in current directory, put .ncm files in there
- Execute `python ncmdump.py`
